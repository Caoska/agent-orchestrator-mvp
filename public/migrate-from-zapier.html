üöÄ Step-by-Step Migration Guide

Step 1: Audit Your Current Zapier Workflows

List all your Zaps and identify:

  • Workflows that need database access
  • Complex conditional logic requirements
  • High email/SMS usage (check your Zapier bill)
  • API integrations that could be simplified

Step 2: Set Up SiloWorker Account

# Sign up and get API key curl -X POST https://api.siloworker.com/v1/auth/signup \ -d '{"email": "you@company.com", "password": "secure123"}' # Get your API key from dashboard export SILOWORKER_API_KEY="sk_live_..."

Step 3: Configure BYOC Credentials

# Add your SendGrid API key (unlimited emails) curl -X POST https://api.siloworker.com/v1/workspace/credentials \ -H "Authorization: Bearer $SILOWORKER_API_KEY" \ -d '{"type": "sendgrid", "api_key": "SG.your_key_here"}' # Add your Twilio credentials (your SMS rates) curl -X POST https://api.siloworker.com/v1/workspace/credentials \ -d '{"type": "twilio", "account_sid": "AC...", "auth_token": "..."}'

Step 4: Create Your First Workflow

# Create agent with database + email workflow curl -X POST https://api.siloworker.com/v1/agents \ -H "Authorization: Bearer $SILOWORKER_API_KEY" \ -d '{ "name": "User Onboarding", "tools": [ { "type": "database", "config": { "connection_string": "postgresql://...", "query": "SELECT * FROM users WHERE email = {{input.email}}" } }, { "type": "conditional", "config": { "condition": "user.plan === \"enterprise\"", "true_path": "premium_onboarding", "false_path": "standard_onboarding" } }, { "type": "sendgrid", "config": { "template_id": "d-abc123", "to": "{{input.email}}", "dynamic_template_data": { "name": "{{user.name}}", "plan": "{{user.plan}}" } } } ] }'

Step 5: Test and Deploy

# Test your workflow curl -X POST https://api.siloworker.com/v1/agents/{agent_id}/run \ -H "Authorization: Bearer $SILOWORKER_API_KEY" \ -d '{"email": "test@example.com"}' # Set up webhook endpoint for production curl -X POST https://api.siloworker.com/v1/agents/{agent_id}/webhook

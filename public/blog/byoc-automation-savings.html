What is BYOC Automation?

BYOC means using your own API keys for services like SendGrid (email) and Twilio (SMS) instead of paying platform markup. Most automation platforms charge 5-10x more than the actual service costs.

Traditional Platform Model

  • Platform buys SendGrid/Twilio credits in bulk
  • Marks up prices 500-1000% for profit
  • You pay the inflated rate with no transparency
  • Limited to platform's feature set

BYOC Model

  • You create your own SendGrid/Twilio accounts
  • Pay actual service rates (often 80% less)
  • Full access to all features and settings
  • Your data stays in your accounts

How to Set Up BYOC

1. Create Your Service Accounts

# SendGrid (Email) 1. Sign up at sendgrid.com 2. Verify your domain 3. Create API key with "Mail Send" permissions 4. Cost: $0 for first 100/day, then $0.0006/email # Twilio (SMS) 1. Sign up at twilio.com 2. Verify your phone number 3. Get Account SID and Auth Token 4. Cost: $0.0075/SMS in US

2. Configure in SiloWorker

# Add your credentials (one-time setup) curl -X POST https://api.siloworker.com/v1/workspace/credentials \ -H "Authorization: Bearer sk_live_your_key" \ -d '{ "sendgrid_api_key": "SG.your_key_here", "twilio_account_sid": "AC...", "twilio_auth_token": "your_token" }'

3. Use in Workflows

# SendGrid tool automatically uses your key { "type": "sendgrid", "config": { "to": "{{user.email}}", "template_id": "d-abc123", "dynamic_template_data": { "name": "{{user.name}}" } } } # Twilio tool uses your rates { "type": "twilio", "config": { "to": "{{user.phone}}", "body": "Order confirmed: {{order.id}}" } }

Common BYOC Questions

Q: Is BYOC more complex to set up?

A: Initial setup takes 10 minutes. After that, it's completely transparent - you just build workflows normally and get charged at your actual rates.

Q: What if I need support?

A: You get support from both SiloWorker (for the platform) and SendGrid/Twilio (for their services). Often better than single-vendor support.

Q: Can I switch back?

A: Yes, you can toggle between BYOC and platform rates anytime. Many customers start with platform rates for testing, then switch to BYOC for production.
